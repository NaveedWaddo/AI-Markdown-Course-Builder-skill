# 📄 Sample Output

This folder contains an example of what the **AI Markdown Course Builder** generates.

The sample is based on a Docker course request:
> *"I want to go from zero to hero in Docker"*

## Files

| File | What it is |
|------|-----------|
| [`01-what-is-docker.md`](./01-what-is-docker.md) | A full topic notes file — Topic 1 of the Docker course |
| [`PROGRESS-TRACKER-sample.md`](./PROGRESS-TRACKER-sample.md) | The PROGRESS-TRACKER.md generated after syllabus confirmation |

## What to look for

Open `01-what-is-docker.md` to see the full notes format:
- Every section from Concept Overview to Quick Notes
- Mermaid architecture and sequence diagrams
- Code examples with anti-patterns
- 6 interview Q&As with model answers
- 2 scenario-based architect-level problems

Open `PROGRESS-TRACKER-sample.md` to see:
- The full 20-topic syllabus with phases → topics → subtopics
- Status tracking table (⬜ / 🔄 / ✅)
- The AI Resume Prompt block — ready to paste into any AI to continue the course
